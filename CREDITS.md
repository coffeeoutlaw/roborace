# Credits

## Robot models (public/assets/robots/)

Four FBX robot models supplied by the project owner (Tripo-generated), each paired
with a basecolor texture downscaled from 8192² to 1024² for the web:

- `cute.fbx` + `cute_basecolor.jpg` — player seat
- `futuristic.fbx` + `futuristic_basecolor.jpg` — Crusher
- `classic.fbx` + `classic_basecolor.jpg` — Prudence
- `screw.fbx` + `screw_basecolor.jpg` — Turbo

Originals live in `assets/models/` (zips). If a model fails to load, the renderer
falls back to a primitive robot built from code.

## Board layouts

Course layouts for Cross, Island, Exchange, Spin Zone, Laser Maze, and Pit Maze
were transcribed from **Spacebug's Robo Rally board recreations**
(https://spacebug705671078.wordpress.com/spacebugs-robo-rally-boards/), which
recreate the official classic RoboRally factory floors. All in-game art is
generated from code (canvas textures); no board images are shipped.

RoboRally was designed by Richard Garfield. This is a non-commercial personal
fan project.
